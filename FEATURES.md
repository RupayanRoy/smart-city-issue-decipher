# ✨ Key Features

### 1. AI Agent Terminal
A conversational interface that guides users through the reporting process. It uses natural language processing to identify if a user is reporting a "Water Leak" or a "Pothole" and asks for missing information like location or photos.

### 2. Global Operations Map
A real-time map overview that clusters city issues. It allows admins to see "hotspots" of activity and citizens to see what's happening in their immediate neighborhood.

### 3. Automated Escalation
To ensure no citizen is ignored, the system automatically monitors "Pending" reports. If an issue isn't addressed within 3 days, the system upgrades its priority to "High" and flags it for immediate Admin attention.

### 4. Verification Loop
Unlike traditional reporting tools, CityCare doesn't close a ticket when a worker says it's done. It moves to a "Completed" state, requiring the Citizen to verify the work, creating a closed-loop accountability system.

### 5. Severe Alert System
Admins can flag specific issues as "Severe Alerts" (e.g., a major gas leak). This triggers a high-visibility overlay for all citizens in the area, ensuring public safety.
