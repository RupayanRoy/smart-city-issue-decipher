# 💻 Technical Stack & APIs

CityCare is built with a modern, performant, and scalable tech stack.

### ⚛️ Frontend Framework
- **React 19**: Utilizing the latest features for a responsive and interactive UI.
- **TypeScript**: Ensuring type safety across the entire application.
- **Vite**: Lightning-fast build tool and development server.

### 🎨 UI & Styling
- **Tailwind CSS**: Utility-first CSS for custom, responsive designs.
- **Shadcn/UI**: High-quality, accessible components built on Radix UI.
- **Lucide React**: A beautiful and consistent icon library.
- **Sonner**: Sleek, non-intrusive toast notifications.

### 📊 Data & Logic
- **Recharts**: Interactive data visualization for the Admin Analytics dashboard.
- **React Leaflet**: Open-source mapping library for issue visualization and location picking.
- **TanStack Query**: Efficient server-state management and caching.

### 🤖 AI & Services
- **CityCare AI Engine**: A custom-built pattern-matching engine for intent detection, category classification, and priority assignment.
- **Mock Backend**: A robust local-storage based database system simulating real-world CRUD operations.

### 🌐 External APIs
- **OpenStreetMap (Nominatim)**: Used for:
  - **Forward Geocoding**: Converting text addresses into Map coordinates.
  - **Reverse Geocoding**: Converting map pins into human-readable addresses.
  - **Tile Server**: Providing the base map layers for the entire platform.
