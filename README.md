# 🏙️ CityCare: Community Impact Platform

CityCare is a next-generation "Smart City" community platform designed to bridge the gap between citizens and local government. It leverages AI to streamline urban issue reporting, provides a tactical command center for administrators, and offers a mission-focused dashboard for field workers.

## 🚀 Vision
To empower every citizen to be a "Community Hero" by providing frictionless tools to report, track, and verify the resolution of urban infrastructure issues, ensuring a safer and cleaner city for everyone.

## 🛠️ Quick Start
1. **Login**: Use the demo accounts provided on the login page.
2. **Citizen**: Report an issue using the AI Assistant or Manual Form.
3. **Admin**: Assign the issue to a worker from the Command Center.
4. **Worker**: Complete the task, upload evidence, and submit the report.
5. **Citizen**: Verify the resolution or re-issue if the problem persists.

## 🏗️ Project Structure
- `src/pages`: Core application views (Citizen, Admin, Worker).
- `src/components`: Modular UI components built with Shadcn/UI.
- `src/backend`: Mock database and business logic services (AI, Analytics, Escalation).
- `src/integrations`: External API configurations.
